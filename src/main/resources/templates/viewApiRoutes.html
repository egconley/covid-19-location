CV19Locator API

For Developers

The CV19Locator API obtains case data from the GitHub of the Johns Hopkins University Center for Systems Science and Engineering.

This public API allows developers to avoid the hassle of having to parse through .csv files and come up with arduous implementations to sort through case data by location.

Postman documentation will be available soon!

Current available endpoints (copy-paste these URLs into your browser or Postman):

http://cv19location.com/API/series/global/confirmed

Obtains time series global data on confirmed cases.

http://cv19location.com/API/series/global/deaths

Obtains time series global data on fatal cases.

http://cv19location.com/API/series/global/recovered

Obtains time series global data on recovered cases.

http://cv19location.com/API/series/US/confirmed

Obtains time series US data on confirmed cases.

http://cv19location.com/API/series/US/deaths

Obtains time series US data on fatal cases.

http://cv19location.com/API/uifcountries

Obtains country codes and population data on countries/provinces/counties.
